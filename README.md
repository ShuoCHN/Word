# 考单词程序4.6
 可以考在英汉列表里的单词
 并可以随机考单词（不重不漏）
 还可以得到错误单词的个数，并总结错误的单词，方便背诵
# 2.0新增功能：
 可以对照英语词和汉语词表的个数，若数量不对应会提示检查
 可以统计出总词数，并在开头汇报
 在开头会汇报所有需要考的单词和对应汉语，方便背诵
 如果单词错误，会提示还剩下多少个单词没有考，用于掌握进度
 每一回合结束会统计出正确率，并汇报
# 2.5新增功能：
 可选择是否进行随机考试（可以按顺序进行考察，便于调试）
 增加了回合数
 改进了退出界面样式
# 3.0新增功能:
